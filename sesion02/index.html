Diferencia entre px y rem como unidades de medida

PX

Los píxeles son unidades fijas y no cambian en función de otros factores por lo tanto un valor en píxeles se ve igual en todas partes, sin importar la configuración del documento o la escala del dispositivo.

REM

Ahora bien, un REM es una medida relativa al tamaño de la fuente del elemento raíz por lo general esta normalmente en el elemento `< html >`

Un ejemplo si el tamaño fuente del elemento raiz es de 16 pixeles, debemos de multiplicar la cantidad de rem por el tamaño fuente para estruturar el tamaño


html {
font-size: 16px; /* 1 rem = 16px */
}
.box {
width: 6.25rem; /* 100px / 16px = 6.25rem */
height: 3.125rem; /* 50px / 16px = 3.125rem */
}
